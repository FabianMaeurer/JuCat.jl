JuCat.jl

JuCat is a package under development with the intention to provide a framework as well a examples for computations in the realm of categories.

Installation

You need to have Julia installed. For reliable results Julia version at least 1.6 is required. To use JuCat do the following:

julia> import Pkg
julia> Pkg.add("https://github.com/FabianMaeurer/JuCat.jl")

Usage

To use JuCat the structures from the OSCAR-System are required. Here a minimal usage Example.

julia> using JuCat, Oscar;
julia> F = FiniteField(5)Galois field with characteristic 5
julia> G = symmetric_group(2)Sym( [ 1 .. 2 ] )
julia> X = gset(G,[1,2,3])Oscar.GSetByElements{Oscar.PermGroup}(Sym( [ 1 .. 2 ] ), ^, [1, 2, 3], Dict{Symbol, Any}())
julia> C = ConvolutionCategory(X,F)Convolution category over G-set with 3 elements.
